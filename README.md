# TypeScript Website – Informatikprojekt

Dies ist ein Informatikprojekt, welches mit **React** und **TypeScript** umgesetzt wurde.

## Technologien

- **React**
- **TypeScript**
- **Vite** 

## Projektstruktur

```bash
.
├── public/               
├── src/
│   ├── assets/       
│   ├── pages/            
│   ├── App.tsx           
│   ├── main.tsx          
│   └── index.css                 
└── README.md             
```
# Setup

1. `git clone https://github.com/Zorilas/Typescript-Website/`
2. `npm install`

# Running
`npm run dev`
